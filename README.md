#Usage
```bash
cd server
pnpm install
pnpm start
cd ...
# Make sure Flutter development env is setup
#  https://docs.flutter.dev/get-started/install
# If you don't want to test it on flutter just open /server/client/index.html on the browser and open inspector to see the logs
# For now, the flutter version is tested on iOS devices
cd flutter
flutter pub get
flutter run
```
